### Welcome the Forms Module

## Form Sections

- Adding a form sections

`Endpoint`

```JS
/api/formSections
```

```JSON
{
    "name": "Form Section Name",
    "sort": 3,
    "form": {"id": "Q9xLAnuFkRLsF"}
}
```

- Updating a form section

`Endpoint`

```JS
/api/formSections
```

```JSON
{"name": "Watts Information",
"sort": 2,
"form":{"id":"USi3thE3EQ5Ey"}}
```
